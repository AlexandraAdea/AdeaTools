KORREKTER BUILD COMMAND FÜR RENDER:

pip install -r requirements.txt && python manage.py collectstatic --noinput && python manage.py migrate && python manage.py create_superuser

WICHTIG:
- Kopieren Sie den kompletten Command oben
- Fügen Sie ihn in Render → Settings → Build Command ein
- Prüfen Sie, dass er NICHT abgeschnitten ist
- Der Command muss mit "create_superuser" enden
