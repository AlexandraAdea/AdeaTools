# PWA Icons

Für die PWA (Progressive Web App) werden folgende Icons benötigt:

- `icon-192.png` (192x192 px)
- `icon-512.png` (512x512 px)

**Erstelle Icons mit:**
- https://realfavicongenerator.net/
- Oder mit Photoshop/GIMP

**Logo-Vorschlag:**
- Buchstaben "AT" (AdeaTools)
- Blauer Hintergrund (#007aff)
- Weiße Schrift

**Temporär:** Platzhalter-Icons verwenden (später ersetzen)
